# Clue Ledger

Psychological suspense needs clue bookkeeping because emotional ambiguity can make evidence feel vague. Track clues so mystery, relationship, and theme return together.

## Ledger Fields

For each clue, record:

- clue name
- first appearance
- who finds it
- how they access it
- what it appears to mean
- what is misunderstood
- who withholds or distorts it
- when it returns
- what it means after return
- whether it points to body, relationship, procedure, or theme

## Evidence Source Test

For each clue, ask:

- How did this person get it?
- Why can police/procedure access it?
- If private, who leaked, preserved, copied, backed up, stole, or delayed it?
- Does the chapter name the source clearly enough?
- Does the source fit prior timestamps?

## Clue Types

### Body Clue

Examples: wound, scar, recording, gesture, posture, medical trace.

Must connect to a mental pressure and later return with changed meaning.

### Document Clue

Examples: contract, message, draft, meeting note, report, record.

Must have plausible storage, access, and modification history.

### Image / Audio Clue

Examples: photograph, surveillance, raw footage, voice note.

Must specify who captured it, who preserved it, and whether it is complete, edited, copied, or imitated.

### Procedure Clue

Examples: door access, vehicle route, temperature log, payment record, call detail.

Must give external pressure and not rely only on protagonist intuition.

### Relationship Clue

Examples: gift, habit, phrase, private joke, repeated gesture.

Must be legible emotionally first and evidentially later.

## Return Pattern

A strong clue returns in three forms:

1. trace: readers notice but cannot decode
2. contradiction: new context makes it troubling
3. reversal: it means something more dangerous than first assumed

## Warning Signs

- A clue appears once and never returns.
- A clue is explained immediately after appearing.
- A private clue has no path into the scene.
- A clue exists only to prove plot, not pressure a relationship.
- A symbol or initials repeat so often they become signage.
