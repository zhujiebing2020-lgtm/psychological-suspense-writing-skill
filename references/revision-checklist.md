# Revision Checklist

Use this after drafting, expanding, or reviewing chapters.

## Chapter Load

Check that the chapter has:

- a complete scene, not only summary
- one emotional confrontation or pressure field
- one concrete object, body trace, or environment detail carrying theme
- one suspense movement
- one character-specific shift
- a chapter ending that opens the next question

If the chapter only advances evidence, expand it with bodily memory or relationship pressure.

If the chapter only explores emotion, add external pressure, a document, a location, or a time constraint.

## Logic

Check:

- How does each character know this information?
- Why does procedure allow this scene?
- Is the evidence source named?
- Are timestamps compatible with prior chapters?
- Does private material have a plausible path into the scene?
- If the protagonist withholds evidence, what are they waiting for or protecting?

## Character Drift

Check:

- The investigator is not a generic detective.
- The witness is not a pure saint.
- The absent/dead/missing person is not only an axis.
- The antagonist is not a simple monster.
- The procedural character does not know things without a source.
- Side characters alter at least one main relationship or clue interpretation.

## Dialogue

Check whether each line belongs to the speaker's pressure system:

- organizer: precise, boundary-making, fact-defensive
- witness: direct, present-tense, refuses fog
- avoider: deflects, jokes, delays, cares instead of answering
- preserver: technical, calm, names desire as repair
- archive keeper: sparse, dry, distrusts neat interpretation
- procedural voice: plain, source-driven, fact pressure

## Theme Control

Prefer:

- an object placed too neatly
- a body arranged too well
- a message saved but unsent
- a care action that feels like restraint
- a professional term hiding violation

Avoid:

- explicit moral equations
- explaining all motives immediately
- body clues as a predictable list
- theme sentences after an image already did the work

## Publication Cleanup

Before giving a publishable version, check:

- repeated phrasing
- overexplaining after strong images
- hook diluted by extra discoveries
- evidence or procedure gaps
- whether the chapter is too short to stand alone
- whether the requested output location needs syncing
